# 【AI系列·第28篇】Vibe Coding进阶——做完整的资金管理系统

文 / Parson资金研习社

---

上一篇用Vibe Coding做了一个"每日资金汇总"脚本——简单、实用、一个下午搞定。

但脚本有局限：数据存在Excel里，多人使用不方便；没有历史记录，查不了过去的数据；没有权限控制，谁都能看所有数据。

这一篇，进阶——用Vibe Coding做一个**完整的资金管理系统**，包含数据库、Web界面、多用户权限。

---

## 一、从脚本到系统：差在哪

先搞清楚"脚本"和"系统"的区别：

| 维度 | 脚本 | 系统 |
|------|------|------|
| 数据存储 | Excel文件 | 数据库 |
| 用户界面 | 命令行 | Web页面 |
| 多人使用 | 不方便（文件冲突） | 支持（并发访问） |
| 历史记录 | 没有 | 有（可查询） |
| 权限控制 | 没有 | 有（按角色分配） |
| 部署方式 | 本地运行 | 服务器部署 |

脚本适合"个人使用、简单场景"。系统适合"团队使用、复杂场景"。

---

## 二、系统架构设计

在动手之前，先想清楚系统要做什么。

### 功能模块

**模块一：数据录入**
- 手动录入子公司资金余额
- 批量导入Excel文件
- 数据校验（格式检查、异常值标记）

**模块二：数据查询**
- 按子公司、币种、日期查询
- 历史数据对比
- 多维度筛选

**模块三：汇总报表**
- 每日资金日报
- 每周资金周报
- 自定义报表

**模块四：预警监控**
- 余额低于阈值预警
- 大额交易异常预警
- 授信使用率预警

**模块五：权限管理**
- 管理员：所有权限
- 资金经理：查看所有数据、生成报表
- 子公司财务：只能查看本子公司数据

### 技术选型

用Vibe Coding做这个系统，技术栈选最简单的：

**后端：** Python + Flask（轻量级Web框架）
**数据库：** SQLite（不需要安装数据库服务器）
**前端：** HTML + Bootstrap（响应式布局）
**部署：** 本地服务器或内网服务器

## 七、第五步：测试和部署

### 测试步骤

**1. 功能测试**

逐个测试每个功能：
- 数据录入：手动录入、批量导入、数据校验
- 数据查询：筛选、排序、导出
- 汇总报表：日报、周报、自定义报表
- 预警监控：规则设置、预警列表、预警处理
- 权限控制：登录、角色检查、权限限制

**2. 并发测试**

模拟多人同时使用：
- 两个人同时录入数据
- 三个人同时查询
- 一个人录入，一个人查询

**3. 数据测试**

用真实数据测试：
- 导入一个月的历史数据
- 检查汇总计算是否正确
- 检查预警是否准确触发

### 部署步骤

**1. 安装依赖**

```bash
pip install -r requirements.txt
```

**2. 初始化数据库**

```bash
python init_db.py
```

**3. 启动服务**

```bash
python app.py
```

**4. 访问系统**

浏览器打开 `http://localhost:5000`

## 九、注意事项

### 1. 数据安全

系统包含敏感的财务数据，必须注意：
- 数据库文件加密存储
- 用户密码用哈希存储（不要明文）
- 传输过程用HTTPS
- 定期备份数据库

### 2. 性能优化

如果数据量大（比如每天上万条记录），需要优化：
- 数据库加索引（日期、子公司、币种字段）
- 查询分页（不要一次加载所有数据）
- 缓存常用查询结果

### 3. 错误处理

系统要能处理各种异常情况：
- Excel文件格式错误
- 数据库连接失败
- 用户输入非法字符
- 并发冲突

### 4. 日志记录

记录所有操作日志：
- 谁在什么时间做了什么操作
- 系统错误和异常
- 预警触发和处理记录

## 十、成本和周期估算

用Vibe Coding做这个系统，大概需要多少时间和成本？

| 阶段 | 时间 | 说明 |
|------|------|------|
| 需求描述 | 1小时 | 把功能需求说清楚 |
| AI生成代码 | 2-3小时 | 分模块生成，边生成边测试 |
| 测试调整 | 1-2天 | 功能测试、并发测试、数据测试 |
| 部署上线 | 半天 | 安装依赖、初始化数据库、启动服务 |
| **总计** | **2-3天** | 从零到可用 |

成本方面：
- 开发成本：几乎为零（用Vibe Coding，不需要雇开发人员）
- 服务器成本：如果用本地电脑，零成本；如果用云服务器，每月几十到几百元
- 维护成本：日常维护由资金团队自己做，不需要IT支持
